# 打工模拟器 v3.0 - 完整版

一款基于控制台的打工经营模拟游戏，让你体验从零开始的创业致富之路！

## 🎮 游戏简介

《打工模拟器 v3.0 - 完整版》是一款用C语言开发的文字经营模拟游戏。玩家将从最基础的打工开始，通过不断完成任务、升级技能、投资理财，最终建立自己的商业帝国！

游戏新增了真实的状态系统和社交系统：
- **饱食度、血量、精神**：打工也要注意身体健康！
- **手机聊天室**：与各式各样的网友互动，但请小心骗子和黑产...
- **意外系统**：工作中可能会发生意外，记得及时恢复


## 🚀 如何开始

### 方法一：直接运行（推荐）
1. 双击 `打工模拟器.exe` 启动游戏
2. 首次运行会自动创建存档文件 `worker_sim_save.dat`

### 方法二：编译运行
1. 确保已安装C语言编译器（如GCC、MinGW）
2. 进入 `源码` 目录
3. 编译命令：
   ```
   gcc 打工模拟器.cpp -o 打工模拟器.exe -lm
   ```
4. 运行编译后的程序

## 🎯 游戏特色

### 核心系统
- **任务系统**：15种不同的打工任务，从出租车司机到企业家
- **技能系统**：10项可升级技能，影响任务效率和收入
- **商城系统**：家具和物品，提供被动收入加成
- **成就系统**：30个成就，记录你的游戏历程
- **合成系统**：物品合成，创造更强装备

### 状态系统（NEW!）
- **饱食度**：每次工作-6，低于20会报警，可购买食物恢复
- **血量**：工作中有概率受伤（伤害5-15），每小时自然恢复5点
- **精神**：工作后随机50-51，影响游戏体验
  - 精神<20：触发幻觉状态，文字变乱码，菜单混乱
  - 精神>78：异常兴奋状态，但有猝死风险！
  - 精神过低或过高都可能导致猝死（可从存档恢复）

### 手机聊天室（NEW!）
- **社交系统**：与10+位AI网友实时聊天
- **人物类型**：
  - 👤 普通人：正常的闲聊对话
  - 🔴 骗子：假装投资顾问，试图诈骗
  - 🟢 黑产：从事灰色产业，可能提供"特殊物品"
  - 💗 同城：提供"放松服务"
- **命令系统**：`@用户名 内容` 可以与指定网友互动
- **特殊互动**：
  - 与黑产聊天提到"OD"会触发特殊事件（降低精神到10，90%猝死几率）
  - 与同城聊天约见面会快速恢复精神（+28精神，+?血量）

### 高级功能
- **季节系统**：春夏秋冬四季轮换，各有不同加成
- **宠物系统**：5种宠物，提供额外收入倍数
- **公司经营**：10级公司升级，获得被动收入
- **股票市场**：5种股票，实时波动，投资理财
- **房产投资**：10处房产，稳定收益来源
- **声望系统**：重置游戏获得永久加成
- **好友系统**：10位NPC好友，社交互动
- **高级会员**：额外特权，登录奖励翻倍

### 随机事件
- 通货膨胀、股市大涨、双倍经验等10种随机事件
- 神秘宝箱、BOSS挑战、节日限定物品和特殊活动
- 抽奖系统，彩票赢取大奖

## 🎮 基本操作

游戏采用数字菜单选择方式：

1. 输入对应数字选择功能
2. 按提示输入操作参数
3. 按任意键继续游戏

### 主要菜单选项
- **1** - 查看玩家状态（含饱食度/血量/精神）
- **2** - 打工赚钱（核心玩法）
- **3** - 商城购物
- **4** - 技能系统
- **5** - 背包物品
- **6** - 任务列表
- **7** - 宠物系统
- **8** - 公司管理
- **9** - 股票市场
- **10** - 房产投资
- **11** - 成就系统
- **12** - 声望系统
- **13** - 好友系统
- **14** - 游戏统计
- **15** - 会员商城
- **16** - 游戏设置
- **17** - 帮助指南
- **18** - 保存游戏
- **19** - 手机（聊天室/购买食物）
- **0** - 退出游戏

### 聊天室用法
1. 进入手机 → 聊天室
2. 输入 `@用户名 你想说的话`
3. 例如：`@路人甲 你好`
4. 特殊词触发隐藏事件：
   - 对黑产说"OD"会触发危险事件
   - 对同城说"约"会触发放松事件

## 💾 存档系统

- 游戏自动保存到 `worker_sim_save.dat`
- 每5分钟自动保存一次
- 每次退出游戏自动保存
- 存档格式：`[4字节版本号] + [Player结构体]`
- 存档版本：v4（新增状态系统和聊天室）
- 包含所有玩家数据（含饱食度/血量/精神）

> 💡 提示：猝死事件后会自动从最近的存档恢复，所以记得常保存！

## 🔧 故障排除

### 问题：游戏界面显示异常
- 解决方案：更新控制台或使用支持ANSI的终端（推荐Windows Terminal）

### 问题：存档损坏
- 解决方案：删除 `worker_sim_save.dat` 重新开始，或使用`.bak`备份恢复

### 问题：无法运行exe文件
- 解决方案：安装 Visual C++ 运行库或使用源码重新编译

### 问题：文字显示乱码
- 原因：可能处于幻觉状态（精神<20），等待1分钟自动恢复
- 或者：检查控制台编码设置

### 问题：角色突然死亡
- 原因：精神过低或过高的猝死事件
- 解决方案：游戏会自动从存档恢复，记得常保存

### 问题：聊天室说不了话
- 解决方案：使用正确的命令格式 `@用户名 内容`
