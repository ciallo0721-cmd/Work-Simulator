# 打工模拟器 v2.0 - 豪华版

一款基于控制台的打工经营模拟游戏，让你体验从零开始的创业致富之路！

## 🎮 游戏简介

《打工模拟器 v2.0 - 豪华版》是一款用C语言开发的文字经营模拟游戏。玩家将从最基础的打工开始，通过不断完成任务、升级技能、投资理财，最终建立自己的商业帝国！

## 📂 文件结构

```
打工模拟器/
├── 打工模拟器.exe          # 游戏主程序
├── kun_save.dat            # 游戏存档文件
├── 修改器/                 # 修改器工具文件夹
│   ├── 修改器.py          # Python修改器脚本
│   ├── kun_save.dat       # 示例存档文件
│   └── readme.txt         # 修改器说明
└── 源码/                   # 游戏源代码文件夹
    └── 打工模拟器.cpp      # C语言源代码
```

## 🚀 如何开始

### 方法一：直接运行（推荐）
1. 双击 `打工模拟器.exe` 启动游戏
2. 首次运行会自动创建存档文件 `kun_save.dat`

### 方法二：编译运行
1. 确保已安装C语言编译器（如GCC、MinGW）
2. 进入 `源码` 目录
3. 编译命令：
   ```
   gcc 打工模拟器.cpp -o 打工模拟器.exe
   ```
4. 运行编译后的程序

## 🎯 游戏特色

### 核心系统
- **任务系统**：10种不同的打工任务，从出租车司机到企业家
- **技能系统**：10项可升级技能，影响任务效率和收入
- **商城系统**：16种家具和物品，提供被动收入加成
- **成就系统**：15个成就，记录你的游戏历程

### 高级功能
- **季节系统**：春夏秋冬四季轮换，各有不同加成
- **宠物系统**：5种宠物，提供额外收入倍数
- **公司经营**：10级公司升级，获得被动收入
- **股票市场**：5种股票，实时波动，投资理财
- **房产投资**：10处房产，稳定收益来源
- **声望系统**：重置游戏获得永久加成

### 随机事件
- 通货膨胀、股市大涨、双倍经验等9种随机事件
- 节日限定物品和特殊活动

## 🎮 基本操作

游戏采用数字菜单选择方式：

1. 输入对应数字选择功能
2. 按提示输入操作参数
3. 按任意键继续游戏

### 主要菜单选项
- **1** - 查看玩家状态
- **2** - 打工赚钱（核心玩法）
- **3** - 商城购物
- **4** - 技能系统
- **5** - 背包物品
- **6** - 任务列表
- **7** - 宠物系统
- **8** - 公司管理
- **9** - 股票市场
- **10** - 房产投资
- **11** - 成就系统
- **12** - 声望系统
- **13** - 游戏设置
- **14** - 保存游戏
- **15** - 帮助指南
- **0** - 退出游戏

## 💾 存档系统

- 游戏自动保存到 `kun_save.dat`
- 每次退出游戏会自动保存
- 存档包含所有玩家数据和进度

### 问题：游戏界面显示异常
- 解决方案：更新控制台或使用支持ANSI的终端

### 问题：存档损坏
- 解决方案：删除 `kun_save.dat` 重新开始游戏

### 问题：无法运行exe文件
- 解决方案：安装Visual C++运行库或重新编译源代码
