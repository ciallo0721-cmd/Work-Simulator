# 打工模拟器 v3.0 - 完整版

一款基于控制台的打工经营模拟游戏，让你体验从零开始的创业致富之路！

## 🎮 游戏简介

《打工模拟器 v3.0 - 完整版》是一款用C语言开发的文字经营模拟游戏。玩家将从最基础的打工开始，通过不断完成任务、升级技能、投资理财，最终建立自己的商业帝国！

## 📂 文件结构

```
打工模拟器/
├── 打工模拟器.exe          # 游戏主程序
├── kun_save.dat            # 游戏存档文件
├── 修改器/                 # 修改器工具文件夹
│   ├── 修改器.py          # Python存档修改器 v2.0
│   └── kun_save.dat       # 示例存档文件
└── 源码/                   # 游戏源代码文件夹
    └── 打工模拟器.cpp      # C语言源代码
```

## 🚀 如何开始

### 方法一：直接运行（推荐）
1. 双击 `打工模拟器.exe` 启动游戏
2. 首次运行会自动创建存档文件 `kun_save.dat`

### 方法二：编译运行
1. 确保已安装C语言编译器（如GCC、MinGW）
2. 进入 `源码` 目录
3. 编译命令：
   ```
   gcc 打工模拟器.cpp -o 打工模拟器.exe -lm
   ```
4. 运行编译后的程序

## 💾 存档系统

- 游戏自动保存到 `kun_save.dat`
- 每次退出游戏会自动保存
- 存档格式：`[4字节版本号] + [Player结构体(37648字节)]`
- 总存档大小：37652 字节
- 存档版本：v3
- 包含所有玩家数据和进度

## 🛠️ 修改器使用

游戏附带一个 Python 存档修改器（v2.0），基于精确的 C 结构体偏移量计算：

### 环境要求
- Python 3.6+

### 使用方法
1. 进入 `修改器` 目录
2. 运行 `修改器.py`
3. 按照提示修改游戏数据

### 支持修改的内容
- 基本属性：等级、经验、坤币、能量、技能点等
- 家具系统：查看/添加/修改家具
- 预设作弊：无限金钱、满级、无限能量、全解锁等
- 高级选项：声望等级、会员状态、成就点等

### 功能特性
- 自动查找存档文件
- 保存前自动创建备份（.bak）
- 存档版本检测与兼容性检查
- 正确支持 `money_t`（8字节）和 `time_t`（8字节）类型

> ⚠️ 注意：修改存档可能影响游戏平衡性，建议先备份原存档

## 🔧 故障排除

### 问题：游戏界面显示异常
- 解决方案：更新控制台或使用支持ANSI的终端（推荐 Windows Terminal）

### 问题：存档损坏
- 解决方案：删除 `kun_save.dat` 重新开始，或使用 `.bak` 备份恢复

### 问题：无法运行exe文件
- 解决方案：安装 Visual C++ 运行库或使用源码重新编译

### 问题：修改器读取数据异常
- 解决方案：确认存档版本为 v3，旧版本存档需要重新开始游戏
